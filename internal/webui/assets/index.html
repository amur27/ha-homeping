Подключение к Home Assistant

Токен хранится в системном хранилище учётных данных и никогда не показывается. Создать: профиль HA → Безопасность → Long-lived access tokens.

Сущности

Режим «Состояния»: уведомление только для перечисленных состояний. Режим «Шаблон»: уведомление для любого значения, {state} заменяется состоянием.
